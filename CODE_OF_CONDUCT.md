# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be kind, assume good faith. Harassment, discrimination, and personal attacks are not tolerated. Disagree about technical decisions — never about people.

## Scope

These rules apply to all project spaces: issues, pull requests, discussions, and any public communication where you represent the project.

## Reporting

If you experience or witness behavior that violates these expectations, report it privately via christophe@lamsitsolutions.com. Reports are confidential. Action will be proportionate to the violation, from a private warning to a permanent ban.
